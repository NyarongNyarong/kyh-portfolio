# Portfolio 2025 GitHub Pages

이 폴더는 PDF를 페이지별 JPG 이미지로 나누어 GitHub Pages에서 보여주기 위한 정적 사이트입니다.

## 구성

- `index.html`: 이미지 포트폴리오 뷰어
- `pages/1.jpg`부터 `pages/32.jpg`: 포트폴리오 페이지 이미지

## GitHub Pages에 올리는 방법

1. 이 `outputs` 폴더 안의 `index.html`, `README.md`, `pages` 폴더를 GitHub 저장소에 올립니다.
2. GitHub 저장소에서 `Settings` -> `Pages`로 이동합니다.
3. `Build and deployment`의 `Source`를 `Deploy from a branch`로 선택합니다.
4. 배포할 브랜치와 폴더를 선택합니다.
   - 저장소 루트에 올렸다면 `/root`
   - `docs` 폴더에 올렸다면 `/docs`
5. 저장 후 잠시 기다리면 Pages 주소가 생성됩니다.

프로젝트 Pages 주소는 보통 `https://username.github.io/repository/` 형식입니다.

## 현재 이미지 상태

- 총 페이지 수: 32장
- 총 이미지 용량: 78,902,680 bytes, 약 75.2MiB
- 각 파일은 GitHub의 100MiB 단일 파일 제한보다 작습니다.
